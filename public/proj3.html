Controls

Keys

w.s.a.d / arrow keys: move player around the screen
w.s.a.d / arrow keys: used to navigate between boxes ( i.e. when selecting starter pokemon or in battle )
j / z: choose new pokemon
j / z: chooses battle action ( i.e. run/fight/ball/pokemon )
k / x: go back from battle action ( i.e. run/fight/ball/pokemon )
u: choose an attack type ( i.e. normal/special )
